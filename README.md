# 인생관리시스템 × 기업교육AI스터디 콜라보 랜딩페이지

인생관리 with AI 앱 워크숍을 소개하는 랜딩페이지입니다.

## 파일 구조

```
LEDsystem_landingpage/
├── index.html          # 메인 HTML 파일
├── css/
│   └── style.css      # 스타일시트
├── js/
│   └── script.js      # JavaScript
└── images/            # 이미지 파일들
    ├── logo.png
    ├── ledsystem01.png
    └── ...
```

## 배포 방법

### GitHub Pages

1. GitHub에 저장소 생성
2. 파일 업로드
3. Settings > Pages에서 배포 설정
4. `https://[username].github.io/[repository-name]` 접속

### Netlify

1. [Netlify](https://www.netlify.com/) 가입
2. "Add new site" > "Deploy manually"
3. 프로젝트 폴더 드래그 앤 드롭
4. 자동으로 배포 URL 생성

### Vercel

1. [Vercel](https://vercel.com/) 가입
2. "New Project" 선택
3. GitHub 저장소 연결 또는 폴더 업로드
4. 자동 배포

## 로컬에서 확인

브라우저에서 `index.html` 파일을 직접 열어 확인할 수 있습니다.
